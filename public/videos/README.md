# Video Files

Place your room tour videos in this folder.

## Required Files:
- `demo-video-1.mp4` - First room tour video (Studio Apartment)
- `demo-video-1.webm` - WebM version of first video (optional, for better browser support)
- `demo-video-2.mp4` - Second room tour video (Interconnecting Apartment)
- `demo-video-2.webm` - WebM version of second video (optional, for better browser support)

## Video Optimization Tips:
- Recommended format: MP4 (H.264 codec)
- Recommended resolution: 1920x1080 (16:9 aspect ratio)
- Recommended file size: Under 10MB per video for web optimization
- Consider creating WebM versions for better compression and browser support
- Videos should be muted and looped for autoplay functionality

Demo placeholders are currently displayed. Replace the demo video files with your actual room tour videos.
